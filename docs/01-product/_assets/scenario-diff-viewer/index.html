Pass two URLs as query parameters to render them side by side:

index.html?left=<URL>&right=<URL>

Typical use:

  • Evolving draft: left = previous-commit raw.githack URL, right = HEAD raw.githack URL of the same path.
  • New draft: left = live ../<slug>.html (the redesigned target), right = draft/<slug>.html.

raw.githack URL pattern: https://raw.githack.com/<org>/<repo>/<ref>/<path>
