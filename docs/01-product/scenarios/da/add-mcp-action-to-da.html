VS Code Add Action Flow

Runs the Add action command against an existing DA project. The toolkit only writes the server URL to .vscode/mcp.json and then opens that file. No tools are fetched and no action manifest is touched during this scenario.

1. Add an Action: pick action type

2. Enter MCP server URL

3. .vscode/mcp.json updated and opened

Recoverable: missing MCP server URL blocks step 3
