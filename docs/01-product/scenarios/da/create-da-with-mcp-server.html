VS Code Create Flow

Guided prompts before the DA project exists. The generated project becomes the precondition for the add-action scenario.

1. Choose project type

2. Choose DA template path

3. Choose action source

4. Choose MCP server type (auto-skipped when only Remote is available)

5. Enter MCP server URL

Recoverable: missing MCP server URL blocks step 6

6. Choose project location

7. Enter app name

8. Project ready for MCP action fetch

.vscode/mcp.json injected into the new project

Lets the developer exercise the MCP server from Copilot Chat in VS Code immediately after project creation, without waiting for provisioning. The CodeLens row exposes ATK: Fetch action from MCP as the entry point into the add-action flow.
